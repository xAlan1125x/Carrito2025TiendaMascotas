# Carrito2025TiendaMascotas
Tienda de Mascotas

El presente proyecto trata de una página de ecommerce, centrada en la funcionalidad del carrito de compras. El mismo permite:
*Mostrar productos con imagen, nombre, precio y botón Agregar.
*Añadir productos al carrito y persistirlos en localStorage.
*Ver un contador de productos en el icono del carrito.
*Abrir/cerrar el panel del carrito.
*Modificar cantidades (+ / −).
*Vaciar carrito o confirmar compra.
*Mostrar subtotal, descuento y total.
*Aplicar códigos de descuento (porcentaje, monto fijo, mínimo requerido).
